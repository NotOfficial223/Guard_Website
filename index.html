Welcome to Guardia!

The moto in Guardia is 'Protection comes first' so thats why we made . . . Guard!












Guard has a lot of moderation and other commands.
This is a full list of all the commands Guard has!


  1. Help - Sends a list of commands!

  2. Avatar - Sends a users avatar!

  3. Ban - Bans a user from a server!

  4. Kick - Kicks a user from a server!

  5. Lockdown - Locks a channel so only admins can speak!

  6. Clear - Clears a certain amout of messages!

  7. Emoji-Steal - Takes an emoji from another server and adds it to yours!

  8. Mute - Mutes a user for a certain amount of time!

  9. Nuke - Clears a channel of **ALL** messages!

  10. Unmute - Unmutes a muted user!

  11. Jail - Makes a user only able to see 1 channel, their jail channel!

  12. Unjail - Makes a jailed user able to see channels again, deletes their jail!

Welcome to Guardia!

The moto in Guardia is 'Protection comes first' so thats why we made . . . Guard!


















Guard has a lot of moderation and other commands.
This is a full list of all the commands he has!


Help - Sends a list of commands!

Avatar - Sends a users avatar!

Ban - Bans a user from a server!

Kick - Kicks a user from a server!

Lockdown - Locks a channel so only admins can speak!

Clear - Clears a certain amout of messages!

Emoji-Steal - Takes an emoji from another server and adds it to yours!

Mute - Mutes a user for a certain amount of time!

Unmute - Unmutes a muted user!

Nuke - Clears a channel of **ALL** messages!

Jail - Makes a user only able to see 1 channel, their jail channel!

Unjail - Makes a jailed user able to see channels again, deletes their jail!
